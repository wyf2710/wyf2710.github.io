    解读数据：导读和热身

    当今世界，各行各业对信息技术的依赖程度在不断加深，每天都会有大量的数据产生。我们常常会感到数据越来越多，但是要从中发现有价值的信息却越来越难。这里所说的信息，可以理解为对数据集处理之后的结果，是从数据集中提炼出的可用于支撑和指导决策的结论性的东西，而从原始数据中抽取出有价值信息的这个过程我们就称之为**数据分析**，它是数据科学的重要组成部分。

    对于想从事数据分析工作的人来说，需要掌握两个部分的技能，一是“数据思维”（统计思维-分析方法-分析模型-业务知识-挖掘算法），二是“分析工具”（SQL-Python-Excel-Tableau-PowerBI）。

    分析工具部分其实是比较容易掌握的，像 SQL 或 Python 这样的编程语言，只要经过系统的学习和适量的练习，大部分人都是可以驾驭的；而像 Power BI、Tableau这样的商业智能工具，更是让我们通过“拖拉拽”操作就能完成数据的可视化并在此基础上产生商业洞察，上手难度会更低一些。相反，数据思维部分的内容对大多数新手来说是不太容易驾驭的，比如“统计思维”，很多人在读书的时候都学习过“概率论”、“统计学”这类的课程，但是当面对实际的业务场景时，却很难将这些知识映射到业务场景来解决现实的问题。

    关于数据分析师

    数据分析师的主要职责一方面是通过数据优化产品和业务，另一方面是通过数据驱动产品和业务。做优化的数据分析师主要是通过对数据的监控和复盘，发现产品和业务中的问题，揭示问题的底层原因并提出自己的建议；做驱动的数据分析师还需要通过对历史数据的建模，为将来产品和业务的发展指明方向。很明显，后者对数据分析师会提出更高的要求，不仅是解读数据的能力，还有建模预测的能力，除外之外就是对产品和业务的理解和认知。大多数招聘门户网站上，HR 在发布招聘需求时，会将很多岗位都称为数据分析岗，但是根据工作性质的不同，我们可以将其分为偏工程的数据治理方向、偏业务的数据分析方向、偏算法的数据挖掘方向、偏产品的数据产品经理等。我们通常所说的数据分析师主要是指业务数据分析师，这个岗位是比较“正统”的数据分析岗，也是招聘数量最多的岗位，这个岗位有时候也被称作数据运营或商业分析，常见的职位描述（job detail）如下所示。

    1. 负责相关报表的输出。
    2. 建立和优化指标体系。
    3. 监控数据波动和异常，找出问题。
    4. 优化和驱动业务，推动数字化运营。
    5. 找出潜在的市场和产品的上升空间。

    根据上面的描述，我们可以了解到作为业务数据分析师，我们的工作不是给领导一个简单浅显的结论，而是要结合公司的业务，完成监控数据、发现异常、深挖原因、探索趋势等工作，资深的数据分析师还要参与指标体系的建设和维护，需要跨部门参与产品的迭代优化、埋点和算法的设计和落地、各种运营数据的复盘等。不管你是用 Python 语言，还是用 Excel、Tableau、SPSS 或其他的商业智能工具，工具只是达成目标的手段，数据思维才是最硬核的东西，从实际业务问题出发到最终发现数据中的商业价值是终极目标。一个合格的数据分析师应该具备以下技能栈。当然并非所有技能都是必需的，但是从合格到优秀再到卓越，我相信这些相关的技能最终都是不可或缺的。

    1. 计算机科学（数据分析工具、编程语言、数据库）。
    2. 数学和统计学（数据基础、统计思维）。
    3. 人工智能（机器学习中的数据挖掘算法）。
    4. 业务理解能力（沟通、表达、经验）。
    5. 总结和表述能力（商业PPT、工作汇报）。

    关于统计学

    在讨论统计学之前，我们先说说概率论。概率论源于赌博游戏，最早从数学角度研究不确定性，寻找随机性背后规律的并不是数学家而是赌徒。意大利文艺复兴时代，百科全书式的学者吉诺拉莫·卡尔达诺（Girolamo Cardano）发表的《论赌博游戏》被认为是第一部论述概率论的著作。到了17世纪的法国，宫廷贵族圈子里盛行一种掷骰子的赌博游戏，游戏规则是玩家连续掷4次骰子，如果没有出现6点玩家赢；如果出现一次6点，则庄家（相当于现在的赌场）赢；后来为了使游戏更加刺激，游戏规则进行了一些调整，玩家用2个骰子连续掷24次，如果不同时出现2个6点，玩家赢，否则庄家赢。在这样的时代背景下，法国数学家帕斯卡（Blaise Pascal）和费马（Pierre de Fermat）创立了概率论，虽然帕斯卡和费马都不是赌徒，但是他们有一些赌徒朋友。两人的研究表明，虽然无法为各种不确定性问题找到一个确定的答案，但是背后依然有规律可循。从18世纪末到19世纪，数学家们开始对概率论产生了兴趣，法国的雅各布·伯努利（Jacob Bernoulli）、皮埃尔-西蒙·拉普拉斯（Pierre-Simon Laplace）和西莫恩·德尼·泊松（Simeon Denis Poisson）等人，德国的约翰·卡尔·弗里德里希·高斯（Johann Carl Friedrich Gauß），俄国的帕夫努蒂·切比雪夫（Pafnuty Chebyshev）、安德烈·马尔科夫（Andrey Markov）等人，都对概率论的发展做出了很大的贡献。其中，雅各布·伯努利发现，概率论的应用场景远远不止赌博游戏，他将他的思考和研究记录下来，写成了《猜度术》一书，书中首次提到了大数定理（在一个随机事件中，随着试验次数的增加，事件发生的频率越趋近于一个稳定值），这个定理在当时的保险公司得到了充分应用。

    小贴士：伯努利家族3代人共产出了8位科学家，我们简单介绍下最有名的三位。雅各布·伯努利于1676年取得了神学硕士学位，但是后来他读了勒内·笛卡尔（René Descartes）、约翰·沃利斯（John Wallis）的著作后，违背了父亲的意愿，兴趣转到了数学和天文学，他在概率方面的贡献突出，以伯努利分布和伯努利大数定理而闻名。约翰·伯努利（Johann Bernoulli）是雅各布·伯努利的弟弟，以最速降线而闻名于世，他是洛必达的私人数学老师，微积分教科书中的洛必达法则其实是约翰·伯努利的研究成果，此外他还教授过被誉为近代数学先驱的数学大师莱昂哈德·保罗·欧拉（Leonhard Paul Euler）。约翰·伯努利的儿子丹尼尔·伯努利（Daniel Bernoulli）在流体力学方面有突出贡献，著名的“伯努利定律”就是他的研究成果。伯努利家族及其后裔在数学、科学、技术、工程、法律、管理、文学、艺术等方面都享有盛名。

    统计学是收集、处理、分析、解释数据并从中得出结论的科学。公元前，古埃及人为了建造金字塔，开始了统计调查活动；古罗马人在很早的时候，就开始了对人口和土地的普查（census）；后来，各国的执政者为了征税和征兵，需要掌握国家的状况。自17世纪以来，为了研究如何进行人口调查，统计学成为了一门正式的学科。统计学的英语单词“statistics”和德语单词“statistik”都来自于拉丁语“status”，而拉丁语中的“status”指的就是（国家的）“状态”和“现状”。然而，一直以来统计学却是一种尴尬的存在，处于各种鄙视链的底端。从数学的角度看，统计学中的数学原理过于肤浅；从应用科学的角度看，统计学太过于数学，跟应用沾不上边。原子核物理学之父欧内斯特·卢瑟福（Ernest Rutherford）曾经说过，“如果你的实验需要统计学，那么你应该再做一个更好的实验”；著名哲学家卡尔·雷蒙德·波普尔（Karl Raimund Popper）也曾经对归纳逻辑进行过强烈的抨击。曾几何时，由于欧几里得、笛卡尔带给人们的完美体系实在太过迷人，导致很多人都忽视了统计思维这一重要的科学思维方式。最近的十年时间，随着大数据和人工智能时代的到来，统计学又以惊人的速度流行起来，因为大数据的应用已经充分证明了经验主义、归纳推理的强大之处；而人工智能实际上也是大数据加上深度学习的归纳方法所取得的成功。有了统计学，我们就能够处有效的处理海量的数据，也能够正确理解数据分析的结果。

    按照统计方法的不同，我们可以将统计学分类为描述统计、推断统计、**贝叶斯统计**等。描述统计是用来描绘和总结观察量基本情况的统计方法，通常会将整理后的数据做成表格或图表，具体包括：探索数据的集中趋势、离散程度、分布规律和相关性。推断统计是研究如何根据样本数据特征推断总体数据特征的方法，在无法获得全量数据的情况下，推断统计就是最为行之有效的方法。贝叶斯统计学的基础是贝叶斯定理，贝叶斯定理将经验和直觉与概率相关联，和人类大脑思考判断的原理十分类似，简单的说就是在获得数据（证据）之后，先前凭借经验和直觉得出的先验概率是可以改变的，重新计算后的概率我们称之为后验概率。

    几个基本概念

    总体和样本

    总体（population）是被研究对象的全体构成的集合，根据其包含的对象数量是否可数可以分为有限总体和无限总体。把总体分为有限总体和无限总体主要是为了判别每次抽样是否独立。对于无限总体，每抽取一个单位并不影响下一次的抽样结果，因此每次抽取可以看做是独立的；对于有限总体，每抽取一个样本后，总体的对象数量就会减少一个，也就意味着前一次抽样会对本次抽样产生影响，每次抽取并不是独立的。

    样本（sample）是从总体中抽取一部分元素构成的集合，构成样本的元素的数量称为样本容量（sample size）。抽样的目的是根据样本提供的信息去推断总体的特征，在很多现实的场景中，我们都无法获取被研究对象的全体，这种情况下就只能通过样本对总体进行推断。

    参数和统计量

    参数（parameter）是用来描述总体特征的概括性数字度量，通常用希腊字母来表示，例如：总体均值通常用 μ 表示，总体标准差通常用 σ 表示。在无法获得全体对象的情况下，总体参数通常是一个未知的常量，正因如此我们才需要抽样，通过样本计算出统计量，再来估计总体参数的值。

    统计量（statistic）是用来描述样本特征的概括性数字度量，它是根据样本数据计算得出的。由于抽样是随机的，因此统计量的值并不确定，它是一个以样本作为其自变量的函数，其本质就是一个随机变量。统计量通常用英文字母来表示，例如：样本均值通常用 x¯ 表示，样本标准差通常用 s 表示。除此之外，还有一些为了做统计分析构造出来的统计量，例如用于假设检验的 z 统计量、 t 统计量、 χ2 统计量、 F 统计量等。

    一些有趣的问题

    下面我先罗列出一些有趣的概率统计问题，欢迎大家在评论区讨论答案，我也会把我自己对这些问题的想法发到评论区。这些问题有的来自于讲解概率统计的书籍，有的来自于数据分析师的面试题，排名和难度都不分先后。

    1. 一个人做一件事出错的概率是0.3，如果做两件无关的事情，他都做对的概率是多少？
    2. 某家庭有两个孩子，问两个孩子都是女孩的概率是多少？
    3. 某家庭有两个孩子，已知其中一个是女孩，问两个孩子都是女孩的概率是多少？（说明：这个问题存在非常大的争议，比较主流的观点认为题目本身的叙述就有问题，大家可以在评论区发表你们的观点。）
    4. 某家庭有两个孩子，已知老大是女孩，问两个孩子都是女孩的概率是多少？
    5. 假设每个人患X病的概率都是0.5，在针对X病的检测中（检测结果不会出错），两个人的混合体液检测呈阳性，两个人都患X病的概率是多少？
    6. 如果在A发生的情况下B发生的概率是0.8，B发生的情况下C发生的概率是0.7，B不发生的情况下C发生的概率是0.3，则在A发生的情况下，B、C至少有一个发生的概率是多少？
    7. 从混有5张假钞的20张百元钞票中随机抽出两张，将其中一张放到验钞机上检验发现是假钞，问抽出的两张都是假钞的概率是多少？（提示：大家可以思考一下，这个问题的本质跟上面的第5题是一样的，还是有什么区别？）
    8. “三门问题”：美国有一个电视节目叫《让我们做笔交易》，节目的参赛者面前有三扇关闭着的门，其中一扇门后面有汽车，另外两扇门后面各有一只山羊，当参赛者选定一扇门但还没有打开它的时候，知道门后情形的节目主持人会开启剩下两扇门中的其中一扇，露出其中一只山羊，然后主持人问参赛者是否需要换另一扇关着的门。请问：换另一扇门是否会增加参赛者赢得汽车的概率？
    9. 一个赌局，获胜的概率是10%，玩家需要玩多少次才能大概率（95%的概率）保证赢一次？（说明：类似的问题还有很多，例如：“一个人参加面试，获得offer或被拒的概率都是50%，且上一场面试是否被拒跟下一场面试的结果无关，他需要参加多少次面试才能大概率获得offer。”）
    10. 假设一个公司门口有10个停车位，该公司有100名员工上班，每名员工早上8点钟之前开车来上班的概率是10%，这些人每天什么时候来公司是随机的，而且彼此无关，也不存在头一天来晚了没有抢到车位第二天就早到的可能性。问早上8点整开车到了公司，停车场还有车位的概率是多少？（提示：大家还可以进一步思考，如果公司门口有100个停车位，该公司有1000名员工上班，其他条件不变，那么刚才的概率是会变大还是变小呢？）

    1、0.49
    2、0.25
    3、0.3（按出生先后顺序的话，男男，男女，女男，女女，已知其中一个是女孩，那就在男女，女男，女女三种情况中判断，就是0.3）
    4、0.5
    5、0.5（混检阳性说明至少有一人患病，在已知至少一人患病的情况下，两个人都患病的概率就放到了另一个人身上？）
    6、0.86（已知A已经发生，问B、C至少有一个发生的概率，那就三种情况，B0.24，C0.06，BC0.56，最后相加0.86）
    7、0.21（这题跟第五题有区别吧，抽出两张，已知一张是假钞，那么两张都是假钞的概率就是4/19）
    8、我觉得有影响，一辆汽车，两只羊，选中汽车概率就是三分之一，之后主持人放一只羊出来，让参赛者再次选择，在一车一羊中选择，选中汽车概率变为二分之一。
    9、每次获胜概率是10%，那么失败概率就是90%，连续29次之后失败概率为4.7%，达到了95%的成功率。
    10、我的理解是要连续10个人在八点前开车来上班才会导致没有车位的情况出现，那就是10%的10次方的概率没有车位。100个停车位的情况下就是10%的100次方？

    最后，让我们用印度著名统计学家卡利安普迪·拉达克里希纳·拉奥（Calyampudi Radhakrishna Rao）的话作为导读的结束：“在终极的分析中，一切的知识都是历史；在抽象的意义下，一切的科学都是数学；在理性的基础上，所有的判断都是统计学”。
