    • win10右键添加文件项

    在Windows 10中，你可以通过修改注册表来在右键菜单中添加新的文件类型。以下是如何操作的步骤（以Typora+md文件为例）：

    1. 打开注册表编辑器

    1. 按下 Win + R，输入 regedit，然后按下 Enter。
    2. 如果弹出用户帐户控制（UAC）提示，点击“是”继续。

    2. 创建新的文件类型

    2. 右键点击 shell 文件夹，选择 新建 > 项，命名为你想要的新文件类型（例如，NewTextFile）。

    3. 在新创建的项下，再右键点击它，选择 新建 > 项，并命名为 command。

    4. 选择 command 项，在右侧的窗口中双击 (默认)，然后输入你希望执行的命令。例如，若想创建一个空文本文件，你可以输入：

    3. 自定义图标（可选）

    你也可以为你的新文件类型指定一个图标：

    1. 在你新建的文件类型项下（例如 NewTextFile），右键点击空白处，选择 新建 > 字符串值，命名为 Icon。

    完成这些步骤后，你就会在右键菜单中看到你创建的选项了。通过点击它，你可以快速创建你需要的文件类型。

    注意事项：

    • 在编辑注册表时，务必小心，因为错误的修改可能导致系统不稳定。建议在修改之前备份注册表。
    • 如果你不熟悉命令行或注册表操作，也可以使用一些第三方工具来简化这一过程，例如 Right Click Enhancer 或 Easy Context Menu。

    这样操作后，你就可以在右键菜单中新建自定义文件类型了！

    下面是 以添加新建md文件为例 另一种方法

    在磁盘的任意位置 新建一个txt文件， 并且录入以下内容：

    @=“Typora.exe” 指定.md文件的运行程序

    @=“Markdown” 指定右键新建的.md文件的默认名字, 也指定右键菜单新建相应选项名为 MarkDown

    编辑好之后，另存时 将后缀名改为.reg文件。

    保存后 双击运行reg文件。

    然后 打开注册表win+R，打开运行对话框输入regedit。

    打开注册表编辑器修改注册表在计算机>HKEY_CLASSES_ROOT下右键查找，输入Typora.exe 查找，查找到后 打开子目录的defaultIcon，右键修改下图标的路径就行。
